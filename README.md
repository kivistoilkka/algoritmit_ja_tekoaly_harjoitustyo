## Asennusohjeet

1. Kloonaa repositorio
2. Asenna riippuvuudet komennolla `poetry install`
3. Käynnistä sovellus komennolla `poetry run invoke start`

Ohjeet ohjelman käyttöön löytyvät [käyttöohjeista](dokumentaatio/kayttoohje.md).
